Co to jest Express.js?

Express.js to lekki i minimalistyczny framework dla Node.js, stworzony w 2010 roku, zaprojektowany z myślą o prostym i elastycznym tworzeniu aplikacji webowych oraz API. Upraszcza obsługę routingu i middleware pozwalając programiście zachować pełną kontrolę nad strukturą aplikacji.

Dziś Express.js jest jednym z najpopularniejszych frameworków w ekosystemie Node.js, wykorzystywany zarówno w prostych projektach, jak i w dużych, skalowalnych aplikacjach webowych, dzięki ogromnej liczbie dostępnych modułów npm.

Zalety

  • Pełna kontrola nad strukturą aplikacji
  • Obsługa wielu równoczesnych połączeń
  • Łatwe tworzenie API i logiki biznesowej
  • JavaScript jako frontend i backend
  • Ogromny ekosystem pakietów npm i middleware
  • Szybkie tworzenie REST API

Wady

  • "Callback hell" w starszych wersjach Node.js
  • Przez jednowątkowość użycie CPU może być wyższe dla niektórych rozwiązań
  • Niektóre funkcjonalności wymagają integracji z dodatkowymi modułami
  • Mniej „gotowych rozwiązań” niż w Rails czy Django
